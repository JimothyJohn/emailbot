Emailbot

Intelligence, Delivered.

A 100% Serverless AWS email assistant powered by Google Gemini 2.5 Flash. It reads, thinks, searches the web, and replies with beautiful HTML.

View on GitHub
⚡️

Serverless Scale

Runs entirely on AWS Free Tier. SES inbound routing heavily restricts spam, while S3 streams payloads directly to ephemeral Lambda invocations.

🧠

Gemini 2.5 Flash

Equipped with Google GenAI SDK. Capable of processing deep multipart email bodies, deducing context, and invoking live web-search tools.

🎨

Elegant Output

Emailbot writes logically structured Markdown, which the Lambda engine parses into meticulously styled HTML tables, lists, and quote blocks before dispatch.

🛡️

Idempotent Execution

Backed by an ultra-fast DynamoDB caching layer. Duplicate `Message-ID` drops are immediately orphaned, preventing automated out-of-office email loops.

The Data Pipeline

Inbound Email → Route 53 MX Record → Amazon SES Ruleset
SES Payload → Amazon S3 Raw Bucket
S3 Trigger → AWS Lambda
Lambda Check → DynamoDB (Message-ID Exist?)
Lambda Extraction → Parse deep MIME Multipart
Lambda Inference → Gemini 2.5 Flash (Web Search Tools Active)
Lambda Render → Convert Markdown to Elegant HTML
Outbound Action → Amazon SES SendEmail (Multipart/Alternative)

Built with AWS SAM, Moto, Pytest, and purely isolated Python code.
